# MixtarRVS AILang Runtime Integration

This folder owns MixtarRVS-specific AILang entrypoints, demos, and integration shims.

AILang itself should remain a generic language/runtime/toolchain repository.

Current imported files:

- `Display/mddm_shell_host.ail` - AILang entrypoint for the MDDM shell C ABI.
- `Examples/mixtar_visual_kernel_demo.ail` - Mixtar visual shell demo built with AILang UI primitives.
